# Rocket admin template

![Rocket admin template preview](resources/media/logo.png)

# Preview

### Demo Site: [[Here]](https://rocket-admin-template.clouw.com)

## Features

- **Tailwind CSS**
- **Alpinejs**

## How to Use

In your terminal, run the following command

```bash
yarn install
```

## Run development server

```bash
yarn dev
```

Visit [`localhost:5000`](http://localhost:5000/index.html) from your browser.
